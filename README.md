# MVVM-Test-App

WPF アプリケーションを MVVM で実装するために学習したことをまとめたリポジトリです。

- Visual Studio 2022
- .NET6
- MahApps.Metro: 2.4.9

MVVM とは関係ありませんが、デフォルトの UI がダサいので MahApps.Metro を導入しています。
